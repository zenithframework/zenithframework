## Part 2: New Project Installer (NEW)

### Goal
Create `zen new <projectname>` command for modern project installation.

### Current State
- No `zen new` command exists
- Users must manually copy/clone project

### Implementation

#### Step 1: Create MakeProject Command
File: `core/Console/Commands/MakeProject.php`

```php
class MakeProject extends Command
{
    public function handle(Container $container, array $arguments): void
    {
        $name = $arguments[0] ?? null;
        
        if (!$name) {
            $this->error('Project name required: php zen new <projectname>');
            return;
        }
        
        // Download/create project structure
        // Copy from skeleton or clone repo
    }
}
```

#### Step 2: Add to CLI
File: `zen`
```php
use Zen\Console\Commands\MakeProject;
```

#### Step 3: Create Project Template
Directory: `installer/` or use GitHub template

#### Usage
```bash
php zen new myapp
# Creates: myapp/
#   - app/
#   - config/
#   - core/
#   - database/
#   - routes/
#   - views/
#   - .env
#   - zen CLI
```
